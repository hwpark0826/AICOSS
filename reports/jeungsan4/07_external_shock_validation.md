# 07. 외부 충격 가설

프로젝트 제공 자료에는 재개발·교통공사·정책·시설 변화의 위치와 발생일이 없습니다. 따라서 매출 변화와 외부 사건의 선후관계를 검증할 수 없습니다. 사건을 추정하거나 외부 검색으로 메우지 않았습니다.

| event_category | verification_status | note |
| --- | --- | --- |
| 재개발·재건축 | not_verifiable | 사업·인허가·공사·이주 일정 데이터가 제공되지 않음 |
| 교통공사·역세권 변화 | not_verifiable | 공사·개통·출입구 변화 일정 데이터가 제공되지 않음 |
| 상권정책·시설 변화 | not_verifiable | 정책·시설 사건의 위치와 시점 자료가 제공되지 않음 |
