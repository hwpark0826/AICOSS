# 증산역 4번 검증 묶음

이 폴더는 증산역 4번의 상대쇠퇴 1위 결과를 반증 가능한 형태로 점검한 자료만 모아 둡니다. 요약 결론은 `09_final_verdict.md`, 재현 가능한 수치와 그림은 `outputs/jeungsan4/`에 있습니다.

## 읽는 순서

1. `00_reproduction_check.md` — 기존 1위 결과 재현
2. `01_missingness_validation.md` — 업종 행 누락과 민감도
3. `02_boundary_validation.md` — 현행 경계와 공간 단위
4. `03_base_year_validation.md` — 기준연도·분기 추세
5. `04_absolute_decline_validation.md` — 공통업종 절대 변화
6. `05_sales_decomposition.md` — 점포·거래·건당매출 분해
7. `06_spatial_displacement_validation.md` — 인접 상권 비교
8. `07_external_shock_validation.md` — 제공 자료로 검증 가능한 외부 충격 범위
9. `08_matched_control_validation.md` — 유사 상권 비교
10. `09_final_verdict.md` — 최종 판정과 한계

## 현장 확인 맥락

현장 확인상 해당 구역은 재개발구역으로 파악됐다. 이는 점포 이탈을 설명할 수 있는 중요한 맥락이지만, 현재 제공된 자료에는 재개발의 위치·시행일·이주일정·점포별 이전 정보가 없다. 따라서 기존 정량 검증에서는 이를 원인으로 확정하지 않고, 추가 확인 항목으로 남긴다.
